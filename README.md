# SVRDD

## Overview

Object detection packaging of SVRDD (Street View Road Damage Dataset), a Beijing city road-surface benchmark with region-wise imagery annotated in VOC and YOLO formats.

- Task: object_detection
- Images: 8000
- Annotations: 20804
- Classes: 7 (Alligator Crack, Longitudinal Crack, Longitudinal Patch, Manhole Cover, Pothole, Transverse Crack, Transverse Patch)
- Annotation format: PASCAL VOC XML (source also includes YOLO TXT labels)
- Viewpoint: wide-view
- Country: China (Beijing)

## Region Coverage

- Chaoyang: 2000 images
- Dongcheng: 1000 images
- Fengtai: 2000 images
- Haidian: 2000 images
- Xicheng: 1000 images
